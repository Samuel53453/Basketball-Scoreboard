# 🏀 Basketball Scoreboard

An interactive basketball scoreboard built with **HTML, CSS, and JavaScript**.
This project allows users to track and update scores for two teams in real-time using simple controls.

Created as a solo project, part of the **Scrimba Fullstack Developer Path**, this project focuses on practicing **DOM manipulation and event handling**.

## ✨ Features

* **Real-time Score Updates** – Increase scores for both Home and Away teams
* **Multiple Scoring Options** – Add 1, 2, or 3 points with dedicated buttons
* **Reset Functionality** – Reset both scores instantly
* **Simple UI Design** – Clean scoreboard layout inspired by real basketball displays

## 🧠 What I Learned

While building this project, I practiced key JavaScript and frontend concepts:

* **DOM Manipulation** – Updating score values dynamically using JavaScript
* **Event Handling** – Using click events to trigger score updates
* **State Management (basic)** – Keeping track of scores for both teams
* **Functions** – Creating reusable functions for updating and resetting scores
* **Separation of Concerns** – Structuring HTML, CSS, and JavaScript properly

## ⚙️ The Process

I started by building the basic HTML structure for the scoreboard layout, including sections for both teams.

Next, I styled the layout using CSS to resemble a real scoreboard. After that, I added JavaScript functionality to handle score updates using button clicks.

Finally, I implemented a reset feature to allow users to restart the game easily.
